# Digital Library Management System

## Oasis Infobyte – Java Development Internship

### Description

A Java Swing based Digital Library Management System with Admin Login and Book Management functionalities.

### Features

* Admin Login
* View Books
* Add Books
* Search Books
* Issue Books
* Return Books
* Logout

### Technologies Used

* Java
* Swing
* Eclipse

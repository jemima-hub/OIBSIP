# Number Guessing Game

## Oasis Infobyte – Java Development Internship

### Description

A Java Swing based Number Guessing Game where the system generates a random number and the user tries to guess it.

### Features

* Random Number Generation
* Swing GUI
* Attempt Limitation
* Score Calculation
* New Game Button

### Technologies Used

* Java
* Swing
* Eclipse

### How to Run

1. Open Eclipse
2. Run GuessNumberGame.java
3. Enter guesses and play

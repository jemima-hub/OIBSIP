# Online Examination System

## Oasis Infobyte – Java Development Internship

### Description

A Java Swing based Online Examination System with login authentication, MCQ questions, timer, auto submit and logout functionality.

### Features

* Login Authentication
* MCQ Based Questions
* Timer
* Auto Submit
* Score Calculation
* Logout

### Technologies Used

* Java
* Swing
* Eclipse
